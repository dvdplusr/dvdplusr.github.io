A programming language (esolang?) homemade by me and based on the Brainfuck language.

> : Move up on the tape.
< : Move down on the tape.
+ : Increase the current byte by 1.
- : Decrease the current byte by 1.
) : Increase the current byte by 10.
( : Decrease the current byte by 10.
. : Print the char value of the current byte.
! : Set the current byte to 0.

Here's an example "Hello world!" program: !)))))))++.)))-.)---.().+++.!)))++.)))))))))---.(++.+++.(++++.(++.(((((((+++.

Have fun!
